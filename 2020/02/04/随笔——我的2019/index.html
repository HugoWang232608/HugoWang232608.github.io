随笔——我的2019

发表于 2020-02-04 更新于 2021-05-30 分类于 随笔 阅读次数： Valine：

  这段时间实在闲得无聊，决定记录一下我的2019。

  其实元旦和除夕的时候我都有想过总结一下我的2019，但是到了那两天又感觉似乎没有什么必要，毕竟我的2019可以说没有一件事是顺利的（指大方向），嗯，话说到这里我还真仔细想了一下，确实如此。

  虽然2019并不顺利，但奇怪的是，我现在回想起我的2019，却没有那种阴暗压抑的感觉，相反，它给我一种明快的基调。想想原因，可能是因为这一年里我做的事，无论是否顺利，皆是出于我内心真实的想法，一种只考虑自己，无需虑及他人（当然也无害于他人）的轻快感。

  当然，前面说的“明快”也好，“轻快”也罢，都是说我现在的想法，真正身处其中时，肯定没有那么轻松。2019绕不开的一件大事就是考研。2019年以前，我说我要考研，多半是出于家人的意愿，家里人是希望我考研的，然而我甚至都没有问过他们为什么希望我读研，不过后来我也没问，因为不需要了，这次是我自己想去挑战一下自己，好久没有这样为一个确切的目标努力过了，当然这个决定里面也有权衡利弊的因素，不过总的来说算是我自己做的决定了。

  虽说考研的决定是我自己做的，但是考研的过程确实不是一个令人愉快的回忆。我是从三月份开始准备的，前期一边上课一边备考，再加上一些其他的事，效率自然不必多说，等暑假有了大片时间，学校又突然加了小学期安排两个实习，暑假又给安排一个实习，那段时间真的被学校安排得有点崩溃，其后又有一些其他事情，总之我真正开始高效率备考大概是九月以后的事情了，到了十一十二月，又有一堆杂七杂八的考试，以及毕设选题（说到这儿不得不吐槽一下我院卡到12月21日的截止日期），后期我一度觉得自己大概考不上了，好在还是及时调整了心态，按部就班地复习了下去。考研那两天就不必多说了，考的人自然知道。。。

  坦白讲，这次考研我未必考的上，甚至可以说大概率考不上，但是即便如此，这次考研我也不是一无所获，首先我知道了当所有事情劈头盖脸朝我砸过来的时候，我该怎么平衡自己的时间，权衡利弊。在考研的过程中，我重新学了一次数学，这总归是有用的，学的英语自然不必多说，不过让我收获最多的还是政治课，以前是很厌恶这些东西的，但是当自己真的静下心来学习的时候，却又觉得很有道理，让我更好地理解这个国家是怎样的一个国家，这个世界是怎样的一个世界。所谓“世界上只有一种英雄主义……热爱生活”，我想这其中大概也有异曲同工之妙。

  这一年里除去考研之外，似乎还有一些称得上是大事的事情，它们穿插在考研的各个阶段，似乎也没什么好说的，总之我的2019就是四个字：率性而为。

  2020的开局似乎也不怎么顺利，不过放假在家里打打游戏搞搞学习聊聊天睡睡觉也没什么不好的，至于接下来的事，如果没有考上的话，是准备工作还是二战，还没有完全决定，唯一能确定的是我要转行。。。

  所有的总结都是为了更好的前进。最后一段话与诸君共勉：

要把所有的夜归还给星河，把所有的春光归还给疏疏篱落，把所有的慵慵沉迷与不前，归还给过去的我。明日之我，胸中有丘壑，立马振山河。

  最后的最后，写下我从有这个手机开始就写在屏幕上的一段话，我觉得特别符合当下的情景：

愿这世界继续热闹

愿我永远是我
